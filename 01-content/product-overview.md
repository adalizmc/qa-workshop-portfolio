# Product Overview
## Nombre del producto
PetStore E-Commerce Platform
## ¿Qué tipo de producto es?
Es una plataforma de comercio electrónico (e-commerce) orientada a la venta de mascotas, alimentos y accesorios para mascotas.
Incluye:
    * Aplicación web para usuarios finales
    * API para gestión de mascotas, inventario y pedidos
## ¿Qué problema resuelve?
Resuelve principalmente la dificultad para encontrar productos para mascotas en un solo lugar, las limitaciones de horario de tiendas físicas y el acceso a las tiendas.
## ¿Cómo genera valor para el negocio?
    * Aumenta las ventas al permitir compras 24/7
    * Amplia el alcance a clientes fuera de una ubicación física
    * Reduce costos operativos (menos dependencia de tiendas físicas)
    * Mejora la experiencia del cliente al facilitar los procesos
## Usuarios principales
- Clientes
- Administradores del sistema
## Funcionalidades principales
- Catálogo de productos y mascotas
- Carrito de Compras
- Integración con funcionalidades bancarias (cobro con tarjeta)
- Gestión de usuarios (Registro, autenticación y perfil de usuario)
- Gestión de Pedidos 
- CRUD en API backend (de Mascotas)
- Gestión de almacenamiento en API backend
